FocusGuard Extension Test Page

1. Extension Installation Check

First, make sure the extension is properly installed:

  • Go to brave://extensions/
  • Enable "Developer mode" (top right toggle)
  • Click "Load unpacked" and select the focusguard-extension folder
  • The FocusGuard extension should appear with no errors

3. Console Debugging

Open DevTools (F12) and check the Console tab for FocusGuard messages when you visit YouTube or WhatsApp.

You should see messages like:

  • FocusGuard: Intention detector loaded on youtube.com
  • FocusGuard: Checking intention for YouTube

4. Expected Behavior

When working correctly:

  1. Visit YouTube → You should see a red "FocusGuard Script Loaded" indicator for 3 seconds
  2. A blue "Test FocusGuard" button should appear for 10 seconds
  3. The intention prompt should appear automatically or when clicking the test button
  4. After setting intention, a timer should appear in the top-right corner
